# Task 002: Build Static Kanban UI

Status: open

## Goal

Implement the Kanban board UI with static sample data.

## Scope

- Add the three columns: `To Do`, `In Progress`, and `Done`.
- Render task cards with title, description, status, and date.
- Add responsive layout behavior.
- Add accessible controls placeholders for future interactions.

## Acceptance Criteria

- The board renders with all three columns.
- Sample tasks appear in the correct columns.
- The UI remains usable on desktop and narrow screens.
- Buttons and form controls have accessible labels.
- No backend or S3 integration is required yet.

## Deploy and Test Checkpoint

Deploy or run locally and verify the static board is visible and responsive.
